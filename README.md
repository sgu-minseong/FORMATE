# 인테리어 견적 자동화 웹 서비스

## 프로젝트 개요
인테리어 업체가 반복적으로 사용하는 단가표와 견적 템플릿을 저장하고, 견적서 작성 시 이를 불러와 빠르게 견적 초안을 만들 수 있는 웹 서비스입니다.

## 문제 정의
인테리어 견적 작성 과정에서는 비슷한 항목과 수량을 반복해서 입력하고, 현장 조건에 따라 수량과 금액을 다시 계산해야 합니다. 특히 엑셀이나 손견적을 사용하는 소규모 업체는 견적 작성 시간이 오래 걸리고 항목 누락이 발생할 수 있습니다.

## 주요 기능
- 업체별 로그인
- 단가표 관리
- 견적 템플릿 관리
- 템플릿 기반 견적서 작성
- 템플릿 없는 조건의 빈 견적서 작성
- 체크한 항목만 견적서 반영
- 자동저장
- 저장 견적 관리
- PDF 확인

## 기술 스택
- React / Vite
- Supabase Auth, Database, RLS
- Vercel
- AI 기반 개발 보조 도구 활용

## AI 활용
AI를 코드 생성뿐 아니라 데이터 구조 설계, 화면 흐름 개선, 예외 상황 점검, 사용자 피드백 반영 과정에 활용했습니다.
